## 나의 미션 목록 조회

| 항목               | 내용                                                         |
| ------------------ | ------------------------------------------------------------ |
| **API Endpoint**   | `GET`/mission                                                |
| **Request Body**   | 필요 없음                                                    |
| **Request Header** | Authorization: accessToken (String)                          |
| **Query String**   | `status` (String)                                            |
| **Query 설명**     | `status`는 미션 상태를 나타내며, 다음 값을 가질 수 있습니다: |
|                    | - `in_progress`: 진행 중인 미션 조회                         |
|                    | - `completed`: 완료된 미션 조회                              |
